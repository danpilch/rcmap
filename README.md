# Relais & Châteaux — Europe

An interactive, clustered map of the ~309 Relais & Châteaux member hotels across 23 European countries. Search by name, place, country, chef, or amenity; filter by country and amenity; and open a rich detail panel for each house with description, dining, amenities, a click-to-enlarge gallery, and directions.

Built with Vite + React + TypeScript and [MapLibre GL](https://maplibre.org/) using key-free [OpenFreeMap](https://openfreemap.org/) vector tiles.

## Run

```bash
npm install
npm run dev      # http://localhost:5173
```

`npm run build` produces a static bundle in `dist/`.

## Data

Property data lives in `src/data/properties.json`. It is generated by scraping the public Relais & Châteaux site, which is a Next.js app embedding full property data in each page's `__NEXT_DATA__` JSON (no API key needed):

```bash
npm run scrape
```

The scraper (`scripts/scrape.mjs`) enumerates the UK properties from the destinations listing, then pulls coordinates, description, rooms, amenities, restaurant/Michelin info, pricing, images, and contact details from each property page. It is throttled and re-runnable. To extend to another country, change the `tags_destinations` filter / destinations URL in the script.

## Structure

```
scripts/scrape.mjs            site -> src/data/properties.json
src/types.ts                  Property schema
src/filters.ts                region order + amenity filter config
src/useFilteredProperties.ts  Fuse.js fuzzy search + facet filtering
src/App.tsx                   layout + shared selection/filter state
src/components/
  Sidebar.tsx                 brand, search, filter chips, results list
  PropertyCard.tsx            list item
  MapView.tsx                 MapLibre map + markers + hover tooltips
  PropertyDetail.tsx          slide-over detail panel
  Lightbox.tsx                full-screen gallery viewer
  icons.tsx                   inline SVG icons
```

Property images are hotlinked from the Relais & Châteaux CDN.
